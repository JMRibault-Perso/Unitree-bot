TROUBLESHOOTING
---------------

"Python is not recognized"
  ‚Üí Reinstall Python and CHECK "Add Python to PATH"
  ‚Üí Then restart Command Prompt

"ModuleNotFoundError: No module named 'fastapi'"
  ‚Üí Run: pip install fastapi uvicorn websockets

"Port 8000 already in use"
  ‚Üí Use different port: python -m g1_app.ui.web_server --port 9000
  ‚Üí Or kill existing: taskkill /F /IM python.exe (all Python processes)

"Cannot connect to robot"
  ‚Üí Is robot powered on?
  ‚Üí Is robot on same WiFi?
  ‚Üí Try: ping [robot-ip]
  ‚Üí Full help: See WINDOWS_SETUP.md

KEY CHANGES
-----------
Before (Linux only):
  sys.path.insert(0, '/root/G1/go2_webrtc_connect')

After (Cross-platform):
  _project_root = Path(__file__).parent.parent.parent
  _webrtc_path = _project_root / "go2_webrtc_connect"
  if _webrtc_path.exists():
      sys.path.insert(0, str(_webrtc_path))

Before (Linux only):
  subprocess.run(['ping', '-c', '1', '-W', '3', ip], ...)

After (Cross-platform):
  if platform.system().lower() == 'windows':
      subprocess.run(['ping', '-n', '1', ip], ...)
  else:
      subprocess.run(['ping', '-c', '1', '-W', '3', ip], ...)
